# Results Archiving Testing - Create
> Verify if the results of an archived election are created whenever the most recent election has been closed for voting successfully.

**Preconditions:**

- Prepare an 'Admin' account
- Link to the application's 'Admin' page
- Link to the application's landing page
- Prepare both a personal computer or a mobile device (use any OS currently at your possession)
- Install a Browser (any type of browser will suffice for this type of test)
- Establish a stable Internet connection
- A pre-defined open election

<br/>

Scenario 1

 | \# | Step | Expected Behavior | 
 |----|------|-------------------|  
 |  1 | Open device | Verify if opened without technical issues successfully | 
 |  2 | Open browser | Verify if opened without technical issues successfully | 
 |  3 | Navigate to the application's 'Admin Login' page | Verify if the pre-supplied link navigates to the application's designated 'Admin Login' page successfully |  
 |  4 | Login credentials | Verify if loged-in successfully |
 |  5 | Navigate to the application's 'Manage Elections' page | Verify if navigation is successful | 
 |  6 | Close Election | Verify if action is successful | 
 |  7 | Navigate to the application's 'Landing Page' | Verify if navigated successfully |  
 |  8 | Download any archived election results | Verify if navigated successfully | 
 |  9 | View Information | Verify if information can be viewed successfully |
