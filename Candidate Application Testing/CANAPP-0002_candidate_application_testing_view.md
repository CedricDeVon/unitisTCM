# Candidate Application Testing - View
> Verify if information about a selected candidate can be viewed successfully.

**Preconditions:**

- Link to the application's landing page
- Prepare both a personal computer or a mobile device (use any OS currently at your possession)
- Install a Browser (any type of browser will suffice for this type of test)
- Establish a stable Internet connection
- A pre-defined open election
- One or more pre-defined election candidates

<br/>

Scenario 1

 | \# | Step | Expected Behavior | 
 |----|------|-------------------|  
 |  1 | Open device | Verify if opened without technical issues successfully | 
 |  2 | Open browser | Verify if opened without technical issues successfully | 
 |  3 | Navigate to the application's 'Landing Page' | Verify if the pre-supplied link navigates to the application's designated 'Landing Page' successfully |  
 |  4 | Open any ongoing elections | Verify if navigated successfully | 
 |  5 | Click on the 'Meet the Candidates' component | Verify if navigated successfully | 
 |  6 | View candidate | Verify if all candidate-related information can be viewed successfully | 


<br/>

**Post-conditions:**

- None at the moment
